## About

Tako Switch is a desktop app for managing and launching AI coding tools (Claude Code, Codex, Gemini CLI, OpenCode, and more) from a single interface — switch provider configs with one click, and control your local coding sessions remotely.

## Acknowledgements

Tako Switch is a fork of **[cc-switch](https://github.com/farion1231/cc-switch)** by [Jason Young (farion1231)](https://github.com/farion1231), released under the MIT License. The original project's clean provider-switching architecture is the foundation of this app. Huge thanks to the cc-switch authors and contributors. 🙏

We build on top of it to add remote control (powered by a self-hosted backend) and Tako-branded integration. See the original at <https://github.com/farion1231/cc-switch>.
